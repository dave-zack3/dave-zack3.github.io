About

Background

I'm a data scientist with a background in defense and national security, where I built analytical models and tools in high-stakes, resource-constrained environments.

I hold a Master's in Data Science from UC Berkeley's MIDS program. My graduate work spanned machine learning, causal inference, data engineering, and interactive visualization, with a growing focus on sports.

I'm now pursuing a transition into sports analytics, bringing rigorous quantitative methods to the problems teams care about most: player evaluation, game strategy, and performance forecasting.

Projects

A selection of data science projects spanning sports analytics, machine learning, and systems engineering.

Football

QB WAR Forecasting

Hierarchical Bayesian model forecasting quarterback pass WAR for the 2024 and 2025 NFL seasons, built for the Las Vegas Raiders Data Science Challenge. Includes SQL feature engineering on play-by-play data, partial pooling across QBs, and a quadratic aging curve.

Python PyMC SQL Bayesian Inference Pandas
Basketball

NBA Score Differential Prediction

LSTM neural network forecasting NBA game point differentials using sequential team performance data. Captures momentum and fatigue over a game sequence, then benchmarked against a feed-forward network and linear regression baseline via cross-validation.

Python TensorFlow LSTM scikit-learn NumPy
Basketball

NBA Shots Through the Years

Interactive visualization exploring how NBA shooting evolved from 2000 to 2022. Users can filter by team and player to trace the three-point revolution, the mid-range decline, and paint efficiency trends, animated across two decades.

Tableau Altair Python HTML Bootstrap
Baseball

What Makes a Good Baseball Team?

Narrative-driven EDA identifying the statistical drivers of MLB team success. Compares run differential, OBP, ERA, and fielding metrics to surface which factors most predict winning, and how those drivers have shifted across seasons.

Python Pandas Seaborn Matplotlib Jupyter
Research

AI Text Detection Experiment

Randomized controlled experiment testing whether priming affects a person's ability to distinguish AI-generated text from human writing. Designed with power analysis; results analyzed via logistic regression and hypothesis testing.

R Logistic Regression Experimental Design Hypothesis Testing
